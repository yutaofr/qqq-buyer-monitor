# Event-Class Defense Boundary Audit

## 2020-like fast cascades with dominant overnight gaps
- **Boundary Class**: `STRUCTURALLY_NON_DEFENDABLE_UNDER_CURRENT_ACCOUNT_CONSTRAINTS`

## 2015-style flash / liquidity vacuum events
- **Boundary Class**: `RESIDUAL_PROTECTION_LAYER_REQUIRED`

## 2018-style drawdown events with partial containability
- **Boundary Class**: `POLICY_LAYER_REMAINS_MEANINGFUL`

## slower structural stress events
- **Boundary Class**: `MODEL_LAYER_REMAINS_MEANINGFUL`

## rapid V-shape ordinary corrections
- **Boundary Class**: `EXECUTION_LAYER_DOMINATES`

## recovery-with-relapse events
- **Boundary Class**: `POLICY_LAYER_REMAINS_MEANINGFUL`
