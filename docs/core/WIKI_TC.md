## 0. 三個輸出，不是一件事

v14 裡有三條不同的決策軌道，它們彼此相關，但透過物理隔離實現防禦：

1. `raw_target_beta`：貝氏後驗期望，引擎的純粹扭力。
2. `target_beta`：考慮熵（泥地）與慣性後的執行位置。掛車的最終朝向。
3. `deployment_state`：新錢進場的策略。

## 1. 制度態先行：系統先壓縮狀態，再解釋週期

### 1.1 什麼是 REGIME
`regime` 指的是**經濟-風險制度態**。它是對目前總經物理狀態的**壓縮標籤**。

### 1.3 四個 REGIME 的經濟含義

| Regime | 經濟階段 | 直覺含義 | 對 QQQ/QLD 的主要含義 |
| :--- | :--- | :--- | :--- |
| `RECOVERY` | 復甦 | 流動性衝擊過去，風險回歸 | 允許重新加碼，地面開始變乾 |
| `MID_CYCLE` | 擴張 | 經濟盈利擴張，未過熱 | QQQ 為主，柏油路面，穩步前行 |
| `LATE_CYCLE` | 末期 | 成長動能衰減，壓力抬頭 | 逐步減弱進攻，地面變濕（泥地化） |
| `BUST` | 衰退 | 信用流動性雙重惡化 | 保護本金，進入深海泥沼，強制煞車 |
