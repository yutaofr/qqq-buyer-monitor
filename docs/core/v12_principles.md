# v12.1 Architecture Principles

> **The "Why" - Core Design Philosophy**

## 1. Orthogonal Reality & Sentinel (v12.1)
- **Macro Matrix (L1-3)**: Discount, Real Economy, Sentiment.
- **Sentinel Core (L4)**: Micro-structure surprisal and divergence arbitrage.
- **Grand Unification**: Merging macro Bayesian inference with micro Gaussian monitoring via the **Generalized Kelly Equation**.

## 2. Information Honesty & JSD
- **Entropy-First**: Quantify model doubt via Shannon Entropy.
- **Divergence-Second**: Quantify "Consensus Breakdown" via Jensen-Shannon Divergence (JSD).
- **Haircut Logic**: `target_beta = (E[beta] * exp(Alignment * JSD)) / Penalty`. High micro-surprisal results in automatic de-risking.

## 3. Physical PIT (Point-in-Time) Integrity
- **Initial Release Only**: Use ALFRED Vintage data to prevent revision bias.
- **Temporal Isolation**: Enforce 7-year IS / 1-year OOS Walk-Forward loops to prevent cycle-specific overfitting.

## 4. Engineering Robustness
- **Tikhonov Regularization**: Maintain physical full-rank covariance to preserve "pain perception" in deep liquidity crises.
- **Topological Smoothness**: Absolute prohibition of step functions. Use Tanh and Softplus for all risk multipliers.
