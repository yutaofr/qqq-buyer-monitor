## 第一章 系统总览

### 1.1 系统定位

这是一个**中期 QLD/QQQ 波段择时系统**。核心功能：

1. **检测流动性制度变点**：使用 BOCPD 识别宏观流动性管道的结构性转变
2. **QLD 波段进出**：平静制度中持有 QLD 获取杠杆收益，持仓周期 3 个月以上
3. **风险退避**：制度恶化时从 QLD 退到 QQQ 或 Cash，保全本金

| 制度 | 操作 | 典型持仓 | 目标年化交易次数 |
|------|------|---------|----------------|
| 平静（$\bar{P} < 0.15$） | 持有 QLD | 3-12 个月 | -- |
| 过渡（$0.15 \leq \bar{P} < 0.50$） | QQQ + 部分 Cash | 1-3 个月 | -- |
| 压力（$\bar{P} \geq 0.50$） | Cash 为主 | 直到制度翻转 | -- |
| **全年合计** | | | **2-4 次** |

**不做日内交易，不做短周期波段。** QLD 的波动率损耗（Volatility Drag）在短周期内会吞噬杠杆收益。只有在 BOCPD 确认流动性制度处于稳态平静期时，长期持有 QLD 的杠杆红利才能充分释放。

### 1.2 核心管线流

```
  慢时钟（周频）           快时钟（日频）
  +-----------+           +----------------+
  | Reserves  |           | ED 加速度       |
  | RRP       |-> l_mac ->| 买卖价差异常    |-> BOCPD -> P_cp_raw
  | TGA       |    (t)    | Fisher(rho)    |   (3D 条件独立)
  | FRA-OIS   |           +----------------+
  +-----------+                  |
                                 v
                          AEMA 滤波 (a_up=0.5, a_dn=0.08)
                                 |
                          熔断器 (P>0.70 -> 绕过)
                                 |
                          双决策映射 -> L_target
                                 |
                          非对称执行死区 -> L_actual
                                 |
                          执行层 -> QLD / QQQ / Cash
```

## 第二章 先验层：Hazard Function

### 2.1 二维正交结构

$$h(r, t) = \lambda_{\text{macro}}(t) \times g(r), \quad r \leq R_{\max} = 504$$

| 维度 | 变量 | 控制 | 来源 |
|------|------|------|------|
| 时间 $t$ | $\lambda_{\text{macro}}(t)$ | 当前宏观环境紧张度 | 慢时钟（Reserves, RRP, TGA, FRA-OIS） |
| 游程 $r$ | $g(r)$ | 当前制度已持续多久 | 纯先验结构，无数据依赖 |

### 2.2 游程长度调制函数

$$g(r) = \begin{cases} 1 + \dfrac{5}{r+1} & r < 63 \quad \text{(初始收敛区)} \\[6pt] 1 & 63 \leq r < 504 \quad \text{(稳态区)} \\[6pt] \text{N/A (硬截断)} & r = 504 \end{cases}$$

### 2.3 硬截断

$R_{\max} = 504$（2 年）。在引擎实现中：`hazard[R_MAX] = 1.0`，强制将该游程的全部概率质量回流到 $r=0$。

| 参数 | 值 | 冻结状态 |
|------|---|---------|
| $R_{\text{stable}}$ | 63 天 | 冻结 |
| $R_{\max}$ | 504 天 | 冻结 |
| $\kappa_{\text{hazard}}$ | 5 | 冻结 |

## 第三章 似然层：三维条件独立发射分布

### 3.1 观测变量

| 维度 | 变量 | 物理含义 | 预处理 |
|------|------|---------|--------|
| $d_1$ | ED 加速度 | 协方差结构变化速率 | 10 日中位数滤波后差分 |
| $d_2$ | 买卖价差异常 | 做市商退出程度 | 截面 Z-score |
| $d_3$ | Fisher($\rho$) | ED 与价差的同步化 | Fisher z-变换 + 20 日滚动 |

### 3.2 联合似然

$$P(x_t \mid r, \theta) = \prod_{d=1}^{3} \text{St}_{\nu_d^{(r)}} \left(x_t^{(d)};\, \mu_d^{(r)},\, \sigma_d^{(r)}\right)$$

条件独立假设：三个维度共享同一游程长度后验，但各自维护独立的 NIG 充分统计量。

### 3.3 NIG 共轭先验超参数

| 维度 | $\mu_0$ | $\kappa_0$ | $\alpha_0$ | $\beta_0$ | 初始 $\nu$ |
|------|---------|-----------|-----------|----------|-----------|
| ED 加速度 | 0 | 5 | 2.5 | 1.5 | 5 |
| 买卖价差异常 | 0 | 5 | 2.5 | 1.5 | 5 |
| Fisher($\rho$) | 0.20 | 5 | 2.5 | 0.09 | 5 |

| 参数 | 值 | 来源 | 可调性 |
|------|---|------|-------|
| $\kappa_0$ | 5 | 5-7 天检测延迟容忍 + 单日噪声抵抗 | 冻结 |
| $\alpha_0$ | 2.5 | $\nu_0 = 5$ 的直接推导 | 冻结 |
| $\mu_0$ | 逐变量无条件均值 | 历史统计 | 年度校准 |
| $\beta_0$ | $1.5 \times \sigma^2_{\text{unconditional}}$ | 历史统计 | 年度校准 |

### 3.4 NIG 在线递归更新（v1.1 修正）

> **v1.1 修正**：v1.0 使用了需要批量统计量 ($S$, $\bar{x}$) 的批处理公式。以下为严格的在线递归形式，与引擎实现逐行对齐。

每个维度 $d$，每个游程长度 $r$。当游程从 $(t-1, r-1)$ 增长到 $(t, r)$ 并吸收新观测 $x_t^{(d)}$ 时：

$$\kappa_t^{(r)} = \kappa_{t-1}^{(r-1)} + 1$$

$$\mu_t^{(r)} = \frac{\kappa_{t-1}^{(r-1)} \cdot \mu_{t-1}^{(r-1)} + x_t^{(d)}}{\kappa_t^{(r)}}$$

$$\alpha_t^{(r)} = \alpha_{t-1}^{(r-1)} + 0.5$$

$$\beta_t^{(r)} = \beta_{t-1}^{(r-1)} + \frac{\kappa_{t-1}^{(r-1)} \cdot (x_t^{(d)} - \mu_{t-1}^{(r-1)})^2}{2 \cdot \kappa_t^{(r)}}$$

当 $r = 0$（新游程启动）：所有统计量重置为先验 $(\mu_0, \kappa_0, \alpha_0, \beta_0)$。

$\nu$ 自然增长：$\nu^{(r)} = 2\alpha^{(r)} = 5 + r$（新游程厚尾宽容，长游程趋高斯敏感）。

引擎实现中的向量化形式（对应第 8 章 8.4 节）：

```python
def _update_nig(self, old_stats, x_t):
    # old_stats: (504, 3, 4) -> [mu, kappa, alpha, beta]
    mu_old    = old_stats[:, :, 0]
    kappa_old = old_stats[:, :, 1]
    x = x_t[np.newaxis, :]  # broadcast (1, 3)

    kappa_new = kappa_old + 1
    mu_new    = (kappa_old * mu_old + x) / kappa_new
    alpha_new = alpha_old + 0.5
    beta_new  = beta_old + 0.5 * kappa_old * (x - mu_old)**2 / kappa_new

    return np.stack([mu_new, kappa_new, alpha_new, beta_new], axis=-1)
```

### 3.5 Fisher($\rho$) 窗口不截断约束（v1.1 新增）

> **架构铁律**：Fisher($\rho$) 的 20 天滚动窗口**禁止**在 BOCPD 检测到变点时进行截断或重置。

**原因**：截断会创建一个从推理层到观测层的反馈环路。

```
[禁止] Fisher(rho) 计算窗口 <-- 变点检测结果 (P_cp)
              ^                         |
              |                         |
              +------- BOCPD 推理 ------+
```

信息流必须是单向的：观测层 -> 推理层。如果推理层的输出（变点检测）反向修改观测层的输入（Fisher 窗口），系统会产生自我指涉的循环依赖，导致：

1. **统计不稳定**：截断后的窗口（如 3 天）计算的相关性方差为 $1/(3-3) = \infty$，直接导致 NIG 更新发散
2. **阈值敏感性**：在什么 $P_{cp}$ 水平截断？0.3？0.5？0.7？每个阈值会产生不同的 Fisher 值，进而影响 BOCPD 的后验，形成不可预测的反馈循环
3. **回测不可复现**：反馈环路使系统路径依赖于数值精度，跨平台结果可能显著不同

**设计保证**：变点发生后，Fisher($\rho$) 的窗口中包含 "旧制度" 数据的部分会在 ~10 天内（窗口的 50%）被自然替换。BOCPD 的 NIG 模型对 Fisher 维度的在线更新会自然吸收这个渐进过渡——这是 AEMA 平滑 + NIG 共轭更新的内在鲁棒性，不需要外部干预。

## 第四章 AEMA 滤波与映射函数

### 4.1 非对称 EMA（v1.2 参数调整）

$$\bar{P}_{t} = \begin{cases} \alpha_\uparrow \cdot P_{cp}^{\text{raw}} + (1-\alpha_\uparrow) \cdot \bar{P}_{t-1} & \text{if } P_{cp}^{\text{raw}} > \bar{P}_{t-1} \\[4pt] \alpha_\downarrow \cdot P_{cp}^{\text{raw}} + (1-\alpha_\downarrow) \cdot \bar{P}_{t-1} & \text{otherwise} \end{cases}$$

| 参数 | 值 | 物理含义 |
|------|---|--------|
| $\alpha_\uparrow$ | 0.50 | 快速响应恶化（保护不打折） |
| $\alpha_\downarrow$ | 0.08 | 极慢接受好转（v1.2: 从 0.18 降低，匹配 3 个月持仓周期） |
| 熔断阈值 | 0.70 | $P_{cp}^{\text{raw}} > 0.70$ 时绕过 AEMA |

> **v1.2 修改理由**：$\alpha_\downarrow = 0.18$ 的半衰期约 3.5 天，会导致短期噪声衰减后 AEMA 快速回落，触发过早的 QLD 回入。$\alpha_\downarrow = 0.08$ 的半衰期约 8.3 天，配合死区和最低持仓期，确保 QLD 退出后至少需要 2-3 周的持续平静信号才能开始恢复杠杆。

### 4.2 双决策杠杆映射

$$L_{\text{target}} = \underbrace{(1 - \bar{P}_t)}_{\text{participation}} \times \underbrace{\frac{\sigma_{\text{target}}}{\hat\sigma_t}}_{\text{sizing}}$$

$$\hat\sigma_t = (1 - \bar{P}_t) \cdot \sigma_{\text{calm}} + \bar{P}_t \cdot \sigma_{\text{stress}}$$

| 参数 | 值 | 治理归属 |
|------|---|---------|
| $\sigma_{\text{calm}}$ | 18% | CRO 年度审计 |
| $\sigma_{\text{stress}}$ | 45% | CRO 年度审计 |
| $\sigma_{\text{target}}$ | 36% ($= 2\sigma_{\text{calm}}$) | CIO 设定 |

### 4.3 执行层资产配比

| 条件 | QLD 权重 | QQQ 权重 | Cash 权重 |
|------|---------|---------|----------|
| $L \geq 1$ | $L - 1$ | $2 - L$ | 0 |
| $L < 1$ | 0 | $L$ | $1 - L$ |

$L \geq 1$ 时使用 QLD（杠杆收益），$L < 1$ 时退出 QLD（规避日频重置损耗）。

### 4.4 非对称执行死区（v1.2 参数调整）

> **设计理由**：系统目标是 QLD 波段持仓 3 个月以上。过频的切换会累积 QLD 的日频杠杆重置损耗（Volatility Drag），吞噬杠杆红利。死区参数从 v1.1 大幅加宽，抑制假复苏触发的 QLD 回入。

**规则**：

$$L_{\text{actual}} = \begin{cases}
L_{\text{target}} & \text{if } L_{\text{target}} - L_{\text{current}} < -\delta_{\text{down}} \\[6pt]
L_{\text{current}} + 0.15 \times (L_{\text{target}} - L_{\text{current}}) & \text{if } L_{\text{target}} - L_{\text{current}} > +\delta_{\text{up}} \\[6pt]
L_{\text{current}} & \text{otherwise (死区内，不调仓)}
\end{cases}$$

| 参数 | 值 | 物理含义 | v1.2 变更 |
|------|---|---------|----------|
| $\delta_{\text{down}}$ | 0.05 | 去杠杆死区（窄）：保护优先，快速退出 QLD | 不变 |
| $\delta_{\text{up}}$ | **0.30** | 加杠杆死区（极宽）：防假复苏触发 QLD 回入 | 0.15 -> 0.30 |
| 恢复系数 | **0.15** | 触发回入后仅恢复差额的 15% | 0.30 -> 0.15 |

**v1.2 参数调整理由**：

- **$\delta_{\text{up}} = 0.30$**（从 0.15 加宽）：QLD 的回入必须极度谨慎。$\Delta L$ 必须超过 0.30 才触发回入——这意味着 AEMA 平滑后的 $\bar{P}$ 需要从高位持续下降相当长一段时间。配合 $\alpha_\downarrow = 0.08$ 的慢衰减，从 $\bar{P} = 0.5$ 恢复到 QLD 区间通常需要 30-40 个交易日。

- **恢复系数 = 0.15**（从 0.30 降低）：即使触发了回入，每次也只恢复差额的 15%。从 Cash 完全回到 QLD 满仓需要连续 15-20 天的确认信号。这自然地将 QLD 回入过程延展到 3-4 周，避免单日跳入。

- **$\delta_{\text{down}} = 0.05$ 不变**：保护不打折。当流动性恶化信号出现时，必须迅速退出 QLD。

```
Delta_L
+0.30 ─────────────────┬──── 极慢恢复（15% 差额）
                        │
  0   ─────────────────┤──── 宽死区（不调仓）
                        │
-0.05 ─────────────────┴──── 立即退出 QLD（全额）
```

### 4.5 最低 QLD 持仓期约束（v1.2 新增）

> **架构铁律**：QLD 波段的最低持仓周期为 63 个交易日（约 3 个月）。

```python
if current_has_qld and days_in_qld < MIN_QLD_HOLD_DAYS:
    # 在最低持仓期内，只有熔断器（P_raw > 0.70）可以触发退出
    # 常规的死区去杠杆信号被忽略
    if p_cp_raw <= CIRCUIT_BREAKER:
        L_actual = L_current  # 保持当前仓位
```

| 参数 | 值 | 物理含义 |
|------|---|---------|
| MIN_QLD_HOLD_DAYS | 63 | QLD 最低持仓期（约 3 个月） |

**唯一例外**：熔断器（$P_{cp}^{\text{raw}} > 0.70$）可以在任何时刻触发 QLD 退出，不受最低持仓期限制。这保留了系统性危机下的紧急逃生通道。

**物理含义**：QLD 的杠杆收益需要足够长的安静波段才能对冲其日频重置产生的波动率损耗。3 个月是 QLD 杠杆红利从波动率损耗中"回本"所需的典型最低持有期。过短的持仓周期下，QLD 的表现几乎确定性地劣于 QQQ。

## 第六章 执行模型

### 6.1 执行价格

**T+1 日开盘竞价（ATO）**。决策在 T 日 16:30 ET 产生，T+1 日 09:30 ET 执行。

### 6.2 市场冲击模型

$$\text{Slippage}(t) = 3\text{bps} + 2\text{bps} \times \frac{\hat\sigma_t}{18\%}$$

### 6.3 净值更新公式（无双重扣除）

**步骤 1：隔夜结算**
$$\text{NAV}_{t+1}^{\text{pre}} = \text{NAV}_t \times (1 + L_t^{\text{old}} \times r_{\text{overnight}})$$

**步骤 2：开盘调仓 + 日内损益**
$$\text{NAV}_{t+1}^{\text{close}} = \text{NAV}_{t+1}^{\text{pre}} \times (1 - C_{\text{true}}) \times (1 + L_t^{\text{new}} \times r_{\text{intraday}})$$

$$C_{\text{true}} = |\Delta L_t| \times \text{Slippage}(t) \quad \text{(仅物理摩擦)}$$

> 执行缺口（Gap）是归因指标，记录在滑点报告中，**不参与 NAV 计算**。

### 6.4 年化成本

| 年份类型 | 物理交易成本 |
|---------|------------|
| 正常（~20 次调仓） | ~8 bps |
| 压力（~40 次 + 1 次熔断） | ~30 bps |

## 第八章 引擎实现规格

### 8.1 数据结构

固定大小 numpy 数组（不使用环形缓冲区）。

| 组件 | 形状 | 大小 |
|------|------|------|
| `run_length_probs` | (505,) | 4 KB |
| `suff_stats` | (505, 3, 4) | 48 KB |
| `g_r` | (505,) | 4 KB |
| **总计** | | **56 KB (L1 缓存)** |

### 8.2 核心循环五步

1. **预测概率**：三维 Student-t 联合密度（向量化 505x3）
2. **Hazard 向量**：$h = \lambda_{\text{macro}} \times g_r$，`hazard[R_MAX] = 1.0`
3. **后验更新**：`new[0] = changepoint`，`new[1:] = growth[:-1]`
4. **NIG 更新**：`new_stats[0] = prior`，`new_stats[1:] = update(old[:-1], x_t)`
5. **$P_{cp}$ 提取**：`new_probs[0]` -> AEMA -> 死区 -> 配比

### 8.3 $R_{\max}$ 截断

隐式实现，无显式 `if` 语句：
- `hazard[504] = 1.0`：强制终止
- `growth[:-1]`：切片自然丢弃 r=504
- `old_stats[:-1]`：统计量同步丢弃

### 8.4 suff_stats 偏移与 INV-4 保护

```python
new_stats = np.empty_like(self.suff_stats)
new_stats[0] = self.nig_priors                        # INV-4: r=0 恒为先验
new_stats[1:] = self._update_nig(self.suff_stats[:-1], x_t)  # 偏移 + 更新
self.suff_stats = new_stats
```

使用 `np.empty_like` + 显式赋值两行，而不是 `np.roll` + 原地修改。
