# SPT: v8.2 CRISIS 血筹码战术现金回补测试计划

> 版本: v8.2
> 状态: Draft
> 日期: 2026-03-26
> 对应 SRD: `docs/v8.2_crisis_blood_chip_srd.md`
> 对应 ADD: `docs/v8.2_crisis_blood_chip_add.md`

## 1. 测试目标

验证 v8.2 新增的 blood-chip crisis override 满足以下要求:

- 只影响新增现金部署，不影响存量 beta 风控。
- 只在经过组合因子确认的危机反转初期触发。
- 不引入未经授权的 `CRISIS` 部署升级。
- 不破坏现有 v8.1 全样本回测稳定性与文档产物。

## 2. 测试分层

### 2.1 单元测试

测试文件:

- `tests/unit/test_deployment_controller.py`
- `tests/unit/test_feature_pipeline.py`
- `tests/unit/test_signal_expectations.py`

核心用例:

#### TC-BC-001 路径 A 触发 override

- 前置条件:
  - `tier0_regime="CRISIS"`
  - `risk_state=RISK_EXIT`
  - `rolling_drawdown=0.18`
  - `liquidity_roc=1.0`
  - `credit_acceleration=-1.0`
  - `twenty_day_return=-0.10`
- 预期:
  - `deployment_state=DEPLOY_FAST`
  - `blood_chip_crisis_override` 出现在 `reasons`

#### TC-BC-002 路径 B 触发 override

- 前置条件:
  - `rolling_drawdown>=0.20`
  - `capitulation_score>=30`
  - `price_vix_divergence=True`
- 预期:
  - `deployment_state=DEPLOY_FAST`

#### TC-BC-003 路径 C 触发 override

- 前置条件:
  - `rolling_drawdown>=0.18`
  - `price_mfi_divergence=True`
  - `near_volume_poc=True`
- 预期:
  - `deployment_state=DEPLOY_FAST`

#### TC-BC-004 高战术压力阻断 override

- 前置条件: 路径 A/B/C 任一成立，但 `tactical_stress_score=80`
- 预期: `deployment_state=DEPLOY_PAUSE`

#### TC-BC-005 非 CRISIS 不进入 override

- 前置条件: `tier0_regime="RICH_TIGHTENING"`，其余条件满足路径 A
- 预期: 不出现 `blood_chip_crisis_override`

#### TC-BC-006 新软特征进入 Class B

- 验证:
  - `price_vix_divergence`
  - `price_mfi_divergence`
  - `short_squeeze_potential`
  - `bond_vol_spike`
  - `near_volume_poc`
  均被分类为 Class B

#### TC-BC-007 expectation surface 支持 crisis override

- 前置条件: 使用 canonical macro fixture，满足路径 A
- 预期: `expected_deployment_state=DEPLOY_FAST`
