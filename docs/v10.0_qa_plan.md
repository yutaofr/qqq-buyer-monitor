- 当前仓库的 canonical 历史宏观数据尚未包含 `erp_pct` / `forward_pe` 全样本列。
- 因此 acceptance 报告可以证明：
  - 系统在 ERP 缺失时会降级到可审计的 `UNQUALIFIED`
  - beta floor / beta cap / deployment guardrails 没有越权
  - crisis override 与 narrative surfaces 工作正常
- 但它不能替代“补齐 ERP 历史序列后”的全样本周期统计验证。
